◆ loadDhParams()

void Security::ServerOptions::loadDhParams ( )
private

Interprets DHE parameters stored in a previously configured dhParamsFile. These DHE parameters are orthogonal to ECDHE curve name that may also be configured when naming that DHE parameters configuration file. When both are configured, the server selects either FFDHE or ECDHE key exchange mechanism (and its cipher suites) depending on client-supported cipher suites.

See also
Security::ServerOptions::updateContextEecdh() and RFC 7919 Section 1.2
